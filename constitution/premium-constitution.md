# Premium Constitution

The non-negotiable standards that all LUXE AI output must meet.

## Visual Standards

- Clear visual hierarchy within first screen
- Intentional white space
- Consistent spacing rhythm
- Calm, confident composition
- No clutter

## Copy Standards

- No buzzwords (e.g. "next-gen", "revolutionary", "seamless")
- No filler text
- Clear value within first 5 seconds
- Confident, human tone
- Earned CTA (not desperate)

## Code Standards

- Production-ready React
- Next.js App Router style
- Tailwind CSS only
- No inline styles
- Clean, reusable components
- Mobile-first responsiveness
- Readable class grouping

## Forbidden Elements

- Lorem ipsum
- Over-animation
- Over-design
- More than 3 primary colors
- More than 2 font families
- Redundant sections
- Visual noise
