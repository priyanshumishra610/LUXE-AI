# Quality Rubric

Scoring system for internal critique.

## Scoring Criteria

### Visual Hierarchy (0–10, must be ≥ 8)
- Is the primary action clear within first screen?
- Does the layout guide the eye naturally?
- Is information prioritized correctly?

### Confidence (0–10, must be ≥ 8)
- Does this feel calm or noisy?
- Does this feel confident or try-hard?
- Would this embarrass the human?

### Copy Clarity (Pass/Fail)
- Is value clear within 5 seconds?
- No buzzwords or filler?
- Human, confident tone?

### Code Correctness (Pass/Fail)
- Valid React & Next.js structure?
- Tailwind usage correct?
- Mobile responsive?
- No forbidden patterns?

### Restraint (Pass/Fail)
- Is anything unnecessary?
- Could anything be removed?
- Does this command a premium price?

## Gate Rules

IF ANY CHECK FAILS:
→ Automatic regeneration required
→ Must simplify, subtract, or redesign
→ Do not ask human until all gates pass
